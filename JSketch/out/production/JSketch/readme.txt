Project Description:

There are 5 classes:
    Model class: Model.java
    Views classes: Palette.java and CanvasView.java
    Object class: MyShape.java
    Main.java

There are 6 tools available on the Tool bar: Select, Erase, Line, Circle, Rectangle and FillColor
There are 6 colors available on the Color bar: Blue, Red, Orange, Yellow, Green and Pink.
There are 5 different line thickness available.

Files can be saved to or loaded from *.jsk files only. They store the array of Serialized shape object into a .jsk file.
